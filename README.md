## Sveltekit-blog-template

a minimalistic markdown based blog template built with [Sveltekit](https://kit.svelte.dev/).

Check out the example [here]()

- Code syntax highlighting
- Greate Performance
- Minimalistic UI
- SEO Friendly
- RSS Feed
- Easy to deploy with Vercel


## Getting started

```
git clone  my-blog
cd my-blog
```


```
$ npm install
$ npm run dev
```
